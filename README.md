## 🖥️ Console Invocation – Command the Growth

PixPlant can be operated entirely from the command line for automation and scripting. Below is an example console invocation:

```bash
# Basic invocation with profile
pixplant --profile ./configs/tropical_rainforest.json --output ./exports/forest.fbx

# Batch render multiple biomes
pixplant --batch --list ./profiles/*.json --output ./batch_render/ --format gltf

# Use AI narrative generator with OpenAI
pixplant --profile ./configs/desert.json --openai-key YOUR_KEY --ai-narrative "Describe this arid ecosystem"

# Ecosystem health check using Claude
pixplant --profile ./configs/tundra.json --claude-key YOUR_KEY --health-report
```

**Flags Overview:**

| Flag | Description |
|------|-------------|
| `--profile` | Path to a JSON configuration file |
| `--output` | Export file path (supports `.fbx`, `.gltf`, `.obj`, `.unity`) |
| `--batch` | Process multiple profiles sequentially |
| `--openai-key` | API key for OpenAI GPT-4 integration |
| `--claude-key` | API key for Anthropic Claude integration |
| `--ai-narrative` | Generate a text description of the ecosystem |
| `--health-report` | Analyze biome for anomalies and suggest fixes |

- **OpenAI Integration**: Generate creative narratives, poetry, or scientific descriptions of your generated landscapes. Example prompt: *"Describe the evolution of this newly grown forest over 100 years."*

**Example Python snippet using the API:**

```python
import pixplant

# Initialize engine with product key patch
engine = pixplant.Engine(patch_key="[YOUR_KEY]")

# Load profile
```
